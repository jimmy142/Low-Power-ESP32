# ULP-ESP32
Ultra low power ESP32 development board by the Circuit Helper

BOM.csv - Bill of materials used in the build, ordered from LCSC.com

ULP-ESP32.pdf - Schematic of the ESP32-S3 development board

ESP32-LP.zip - Gerber export from EasyEDA of the development board

Tx_S3b.ino - Arduino code for the client (goes on the ESP that reads the BMP sensor and transmits the data)

base_s3b.ino - Arduino code for the base station (goes on the ESP that receives sensor data and creates web page showing data)
